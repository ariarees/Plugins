## Plugins

| Plugin | Description | Docs |
|--------|-------------|------|
| **DoughminationAPI** | Core library - ban system, jail API, cross-plugin events | [📖](https://modding.doughmination.co.uk) |
| **Plural** | Plural system support with message proxying | [📖](https://modding.doughmination.co.uk) |
| **Doughmiantioncord** | Flight, vein mining, spawn eggs, potions, teleports | [📖](https://modding.doughmination.co.uk) |
| **WingSync** | Discord whitelist sync with ban integration | [📖](https://modding.doughmination.co.uk) |
| **JailedDough** | Jail system with timed releases | [📖](https://modding.doughmination.co.uk) |
| **StatsEnd** | Dragon event tracking and achievements | [📖](https://modding.doughmination.co.uk) |

## Quick Start

```bash
# Build all
./gradlew build

# Build specific plugin
./gradlew :pluginname:shadowJar
```

**Installation:** Download from [Doughmination Modding](https://modding.doughmination.co.uk)
